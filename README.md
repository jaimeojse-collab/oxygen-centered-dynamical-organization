# Oxygen-Centered Dynamical Organization of Chemical Space

This repository accompanies the open-science preprint:

"Oxygen-Centered Representations Reveal a Candidate Dynamical Organization of Chemical Space"

## Overview

This project explores whether oxygen-centered representations of chemical space may reveal an underlying dynamical organization.

Using a minimal Endogenous Reachability Collapse (ERC) framework, the notebook constructs a reduced embedding of the periodic table and evaluates:

- Emergence of a dominant dynamical axis (PC1)
- Variation in recoverability-related observables
- Unsupervised regime structure along the axis
- Relationship to oxygen-relative geometry
- Partial correspondence with physical observables (diatomic vibrational frequencies)
- Residual structure relative to oxygen
- Robustness under perturbations
- Dependence on anchor element choice

The analysis is designed as a minimal, hypothesis-generating framework, not as a predictive physical model.

## Reproducibility

All required input files are included in the data/ folder.

The notebook:
- uses repository-relative paths
- does not require external storage (e.g., Google Drive)
- can be run directly in Google Colab or a local Python environment

All preprocessing steps, transformations, and derived quantities are implemented explicitly to ensure reproducibility.

## Main files

- ERC_Oxygen_Vibrational_Modeli_github_pynb.ipynb — main analysis notebook  
- data/ — input datasets  
- figures/ — generated figures used in the manuscript  
- Preprint PDF — full methodological and interpretative context  

## Scope and limitations

- The ERC framework is implemented in a minimal form, focusing on coarse observables of dynamical accessibility  
- Correlations with physical observables are weak and based on limited datasets  
- Results should be interpreted as preliminary and indicative, not conclusive
